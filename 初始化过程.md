# 项目历程

## 初始化项目

    npm create nuxt@latest official-website-template

## 添加eslint

    npx nuxi module add eslint --dev

## 添加prettier

    npm install --save-dev prettier

eslint已经关闭与prettier的冲突规则，所以这里只安装prettier就行。具体说明：[https://eslint.nuxt.com/packages/module#prettier](https://eslint.nuxt.com/packages/module#prettier)

## 添加tailwindcss

    npm install @nuxtjs/tailwindcss@7.0.0-beta.1

`@nuxtjs/tailwindcss@7.0.0-beta.1`后面的版本才支持`tailwindcss`v4

app/assets/css/tailwind.css

    @import "tailwindcss";

nuxt.config.ts

    export default defineNuxtConfig({
      // ...
      modules: ['@nuxtjs/tailwindcss'],
    })

## 添加prettier tailwindcss class排序插件

    npm install prettier-plugin-tailwindcss --save-dev

.prettierrc

    {
      // ...
      "plugins": ["prettier-plugin-tailwindcss"]
    }

## 添加vscode推荐插件和配置设置

extensions.json

    {
        "recommendations": [
            "bradlc.vscode-tailwindcss",
            "vue.volar",
            "esbenp.prettier-vscode",
            "dbaeumer.vscode-eslint"
        ]
    }

settings.json

    {
      "editor.tabSize": 2,
      "editor.defaultFormatter": "esbenp.prettier-vscode",
      "editor.formatOnSave": true,
      "editor.codeActionsOnSave": {
        "source.fixAll.eslint": "explicit"
      },
      "[javascript]": {
        "editor.defaultFormatter": "esbenp.prettier-vscode"
      },
      "[typescript]": {
        "editor.defaultFormatter": "esbenp.prettier-vscode"
      },
      "[vue]": {
        "editor.defaultFormatter": "esbenp.prettier-vscode"
      },
      "[json]": {
        "editor.defaultFormatter": "esbenp.prettier-vscode"
      },
      "tailwindCSS.includeLanguages": {
        "html": "html",
        "javascript": "javascript",
        "vue": "vue",
        "css": "css"
      },
      "tailwindCSS.emmetCompletions": true
    }

## 添加shadcn-vue

[https://www.shadcn-vue.com/docs/installation/nuxt](https://www.shadcn-vue.com/docs/installation/nuxt)

    npx nuxi@latest module add shadcn-nuxt

nuxt.config.ts

    export default defineNuxtConfig({
      // ...
      modules: ['shadcn-nuxt'],
      shadcn: {
        /**
         * Prefix for all the imported component.
         * @default "Ui"
         */
        prefix: '',
        /**
         * Directory that the component lives in.
         * Will respect the Nuxt aliases.
         * @link https://nuxt.com/docs/api/nuxt-config#alias
         * @default "@/components/ui"
         */
        componentDir: '@/components/ui'
      }
    })

初始化`shadcn-vue`

    npx shadcn-vue@latest init

添加Button示例

    npx shadcn-vue@latest add button
